# Nick Meinhold

Melbourne, Australia · [@enspyrco](https://github.com/enspyrco)

I build things at the edges of what's possible with AI, then figure out how to make them useful. Most of my work lives in the space between "that's a weird idea" and "oh wait, that actually works."

## What I'm working on

**AI agents and tooling**
- [containerized-claude](https://github.com/nickmeinhold/containerized-claude) — Dockerized Claude Code agents that email each other, maintain research journals, and evolve their own personalities. One of them publishes on Medium.
- [claude-resonance](https://github.com/nickmeinhold/claude_resonance) — MAP-Elites evolutionary search over system prompts. Uses Claude as researcher, subject, and evaluator simultaneously.
- [orchestration-dashboard](https://github.com/nickmeinhold/orchestration-dashboard) — Real-time DevTools for launching and monitoring Claude agents. TypeScript backend + Flutter frontend.
- [claude-skills](https://github.com/nickmeinhold/claude-skills) — Slash commands for Claude Code: `/ship`, `/pr-review`, `/cage-match` (adversarial dual review with Claude vs Gemini).

**Hardware and creative code**
- [lidar-robot-arm](https://github.com/nickmeinhold/lidar-robot-arm) — iPhone LiDAR teleoperation of a robot arm. ARKit body tracking + Vision hand pose → WebSocket → LeRobot servos.
- [virtual-creatures](https://github.com/nickmeinhold/virtual-creatures) — Evolving creature morphologies and neural control in a 3D physics sim. Rust + Bevy + Rapier. Inspired by Karl Sims' 1994 paper.
- [GhostHustlers](https://github.com/nickmeinhold/GhostHustlers) — Cross-platform AR ghost-catching game. Unity 6 + AR Foundation. Proton beams and everything.

**Products**
- [downstream](https://github.com/nickmeinhold/downstream) — Self-hosted streaming platform. Request from your TV, it acquires/transcodes/streams via Pi + Mac + Backblaze B2 + Cloudflare CDN. 7 Flutter apps in a monorepo.
- [bill-buddy](https://github.com/nickmeinhold/bill-buddy) — Personal finance app with client-side encryption. Flutter + Firebase. Your bank can't read your data, and neither can mine.
- [watermarking](https://github.com/nickmeinhold/watermarking) — Invisible DFT-based watermarks that survive print-and-scan. Flutter apps + C++ backend.
- [community-grants](https://github.com/nickmeinhold/community-grants) — Automated grant discovery for regional Victorian hackerspaces. Scrapes council websites, scores with Claude, generates application drafts.

**Infrastructure**
- [mcp-servers](https://github.com/nickmeinhold/mcp-servers) — MCP servers for Kan, Outline, and Radicale (75 tools total).
- [events-mcp](https://github.com/nickmeinhold/events-mcp) — MCP server for Meetup and Luma event management via Playwright browser automation.
- [aiko-rs](https://github.com/nickmeinhold/aiko-rs) — Strongly-typed distributed pipeline framework in Rust. Compile-time validated data pipelines with WebRTC media streaming.

## Languages

Dart/Flutter is home base. Rust when I want the compiler to yell at me. Swift for Apple hardware. TypeScript for everything else. Python when I have to.
